Learn More About Me!

Hi, my name is Kayleen Do, and one of my favorite hobbies is doing gel nails! I am a computer science student at the University of Virginia, and was looking for a personal project to work on, so I decided to combine my hobby with my coding skills and create a mock website for a nail salon. All the nail art pictured are nails that I've done myself, which you can check out on the gallery page.

I used HTML and CSS to desing the layout and styling for the webpage, and JavaScript to code the functionalities of the site, like zooming into the gallary photos and connecting the contact form to a Google Spreadsheet.

Feel free to explore the site and don't hesitate to reach out if you have any questions!
